# Stonewall

This project contains a conceptual example of an iOS app that uses NFC to trigger distraction blocking logic. The SwiftUI code can be found in the `DistractionBlocker` directory.

## Building
1. Open Xcode and create a new iOS App project.
2. Replace the autogenerated files with the Swift files in the `DistractionBlocker` folder.
3. Enable the **Near Field Communication Tag Reading** and **Network Extensions** capabilities in the target settings.
4. Build and run on a physical iPhone to test NFC scanning.

The `BlockingManager` file includes a placeholder for activating a DNS or content filter using a Network Extension.
